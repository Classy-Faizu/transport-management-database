# Transport Management Database System

## Overview

This project involved designing and developing a transport management database system as part of a team project. The system was designed to support transport operations by managing inventory, driver assignments, and related transport records.

## Technologies Used

- Oracle Database
- SQL
- Database Design

## Features

- Inventory management
- Driver assignment management
- Transport record management
- Conceptual, logical, and physical database design

## Learning Outcomes

- Database modelling
- SQL database design
- Team collaboration
- Database implementation using Oracle
## Project Documentation

This project was completed in three phases:

- **Phase 1:** Requirements analysis and system planning.
- **Phase 2:** Conceptual, logical, and physical database design.
- **Phase 3:** Database implementation, SQL development, and testing.
